# 리스트
리스트는 데이터를 순서대로 한줄로 저장하는 구조이다.

### 단순 연결리스트

아래는 단순한 노드의 구조체 구성과 생성 방법이다.
```c
typedef struct node{
    int elem;
    struct node* link;
}Node;
```
###### 노드 생성 방법
```c
Node* getnode(){
    Node *NEW;
    NEW= (Node*)malloc(sizeof(Node));
    return NEW;
}
```

###### 단순리스트의 탐색연산

```c
Node *search(Node *head, int x) {
	Node *p;
	 p = head;
	 while (p != NULL) {
		 if (p->data == x) return p;
		 p = p->link;
	 }
	 return p;
}
```

###### 단순리스트의 삽입연산

```c
void insert_node(Node **head, Node *p, Node *new_node) {

	if (*head == NULL) {
		new_node->link = NULL;
		*head = new_node;
	}
	else if (p == NULL) {
		new_node->link = *head;
		*head = new_node;
	}
	else {
		new_node->link = p->link;
		p->link = new_node;
	}
}
```

###### 단순리스트의 삭제연산

###### 2개의 단순리스트 병합

``` c
Node *concat(Node *head1, Node *head2) {

	ListNode *p;
	if (head1 == NULL) return head2;
	else if (head2 == NULL) return head1;
	else {
		p = head1;
		while (p->link != NULL) p = p->link;

		p->link = head2;
		return head1;
	}

```

###### 단순리스트 역순으로 정리하기

``` c
Node *reverse(Node *head) {
	Node *p, *q, *r;
	p=head;
	q=NULL;

	while (p!=NULL) {
		r=q;
		q=p;
		p=p->link;
		q->link=r;
	}
	return q;
}
```

### 이중 연결리스트

이중연결리스트는 헤더와 트레일러를 사용해서 관리하면 편하다. 

###### 노드 구조체 정보
``` c
typedef struct node{
    int elem;
    struct node* next;
    struct node* prev;
}Node;
```

###### 헤더 트레일러 구조체 정보

``` c
typedef struct ht{
    struct node* Header;
    struct node* Trailer;
}HT;
```

###### 이중리스트 초기화

``` c
void init(HT* x)
{
    x->Header=(Node*)malloc(sizeof(Node)*1);
    x->Trailer=(Node*)malloc(sizeof(Node)*1);
    x->Header->next=x->Trailer;
    x->Trailer->prev=x->Header;
    return;
}
```

###### 이중리스트의 탐색연산

```c
Node *search(HT *x, int x) {
	Node* p=x->Header->next;
    
    while(p!=x->Trailer)
    {
        if(p->elem==x)
            return p;
    }
    return NULL // 못찾으면 NULL반환
}
```

###### 이중리스트 삽입

``` c
void addNode(HT* x, int rank, int elem)
{
    //1번째 rank는 헤더의 바로 다음이다.
    Node* p=x->Header;
    Node* pnext;
    //그래서 첫번째rank이면 for문을 돌 필요가 없다.
    for(int i=1;i<rank;i++)
    {
        //rank가 초과일때 exception
        if(p==x->Trailer)
            return;
        p=p->next;
    }
    pnext=p->next;
    NEW= (Node*)malloc(sizeof(Node)*1);
    
    p->next=NEW;
    pnext->prev=NEW;
    NEW->prev=p;
    NEW->next=pnext;
   return;
}
```
