# 리스트
리스트는 데이터를 순서대로 한줄로 저장하는 구조이다.

### 단순 연결리스트

아래는 단순한 노드의 구조체 구성과 생성 방법이다.
```{.cpp}
typedef struct node{
    int elem;
    struct node* link;
}Node;

Node* getnode(){
    Node *NEW;
    NEW= (Node*)malloc(sizeof(Node));
    return NEW;
}
```
