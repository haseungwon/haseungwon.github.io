# 집합

연결리스트에 의해 구현되는 집합이 더 광법위하게 사용 가능하다.

데이터 구조의 보존 여부에 따라 파괴적(Destructive), 비파괴적(Nondestructive)으로 나눠진다.

단일연결리스트로 구현해보자.


``` c
typedef struct node{
    int elem;
    struct node* next;
    struct node* prev;
}Node;


```

### Union(B) 합집합

이중연결리스트로 구현된 집합 A 와 B가 있다고 가정하면 비파과적 합집합은 다음과 같이 구한다

``` c
Node *Union(Node *A, Node*B)
{
    Node *p;
    if(A==NULL && B==NULL)
        return NULL;
    p= (Node*)malloc(sizeof(Node))l
    if(A==Null)
    {
        p->elem = B->elem;
        p->next = Union(A,B->next);
    }
    else if(B==Null)
    {
        p->elem = A->elem;
        p->next = Union(A->next,B);
    }
    else
    {
        if(A->elem<B->elem)
        {
            p->elem = A->elem;
            p->next = Union(A->next,B);
        }
        else if(A->elem>B->elem)
        {
            p->elem = B->elem;
            p->next = Union(A,B->next);
        }
        else
        {
            p->elem = B->elem;
            p->next = Union(A->next,B->next);
        }
    }
    return p;
 

}

```

### Intersection(B) 교집합

이중연결리스트로 구현된 집합 A 와 B가 있다고 가정하면 비파과적 교집합은 다음과 같이 구한다

``` c
Node *intersection(Node *A, Node *B)
{
    Node *p;
    if(A==NULL||B==NULL)
        return NULL;
     if(A->elem<B->elem)
        return intersect(A->next,B);
    else if(A->elem>B->elem)
        return intersect(A,B->next);
    else
    {
        p=(Node*)malloc(sizeof(Node));
        p->elem = A->elem;
        p->next = intersect(A->next,B->next);
    }
    return p;
    
}
```
